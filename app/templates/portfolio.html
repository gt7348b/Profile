In addition to the overall JTC Analytics site itself (contructed with HTML, AngularJS, and utilizing the Bourbon and Neat sass libraries), the following projects represent a selection of work by John Crocker.

J & J's Blog

Project Description:

This site deployed on git-hub was jointly developed with Joanna Edgell. This site is deployed on Github and utilizes: Parse, MomentsJS, and Google Fonts.

Exploring the National Transit Database

Project Description:

An application that examines the Monthly Adjusted Data from the National Transit Database to visually display selected data elements. Deployed on Git-hub, this application utilizes: HTML, AngularJS, d3JS and data from the National Transit Database.

Music

Project Description:

An application that displays a list of music deployed on Heroku using: HTML, Parse, and Bourbon.
